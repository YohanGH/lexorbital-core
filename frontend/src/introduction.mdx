## Installation

Le projet utilise React 19, TypeScript, Vite et Tailwind CSS.

```bash
pnpm install
```

## Structure des composants

Les composants sont organisés selon l'architecture atomique :

- **Atoms** : Composants de base (Button, Card, Link, Typography)
- **Molecules** : Composants composés (LanguageSelector, VersionButton)
- **Organisms** : Composants complexes (Header, Footer)
- **Pages** : Pages complètes de l'application

## Utilisation

Chaque composant est documenté avec ses props, variantes et exemples d'utilisation.
Utilisez les contrôles dans le panneau pour tester différentes configurations.

## Versions

Le système supporte le versioning des composants et des pages via le système de
versioning intégré. Consultez la section "Versioning" pour plus d'informations.
