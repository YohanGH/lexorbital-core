# LexOrbital BackRing

Backend API of the LexOrbital Meta-Kernel.

## Endpoints

- `GET /health` - Service status
- `GET /modules` - List of modules (mocked for POC)

## Development

```bash
pnpm install
pnpm dev      # Development with hot-reload
pnpm build    # Production build
pnpm start    # Run in production
```
